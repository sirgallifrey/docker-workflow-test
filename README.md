# docker-workflow-test
A test setup of a docker node dev/prod env

How to install

```sh
# dependencies
git -v
docker -v
docker-compose -v
# get the code
git clone https://github.com/sirgallifrey/docker-workflow-test.git docker-workflow-test && cd $_
# build the container
docker-compose build
#execute containers
docker-compose up
# up and running
```
